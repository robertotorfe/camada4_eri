a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git es un sistema de control de versiones distribuido que, como ya hemos introducido, fue creado por Linus Torvalds en 2005 para ser utilizado en el desarrollo del kernel de Linux. 

b. ¿A quién pertenece actualmente Github y por qué?
Pertenece a Microsoft desde 2018, cuando compró la compañia por 7500 millones de dólares.

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
Existen diferentes formas, usando extensiones en los IDE, aplicaciones con interfaz grafica e incluso se puede trabajar sobre la misma plataforma de Github
