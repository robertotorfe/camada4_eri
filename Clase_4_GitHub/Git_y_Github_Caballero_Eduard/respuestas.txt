a. ¿Quién inventó el sistema de control de versión Git y por qué?
    Lo invento Linus Torvalds, Originalmente, la comunidad de desarrolladores de Linux utilizaba BitKeeper, un sistema de control de versiones distribuido y comercial, sin embargo, el hecho de ser un software propietario y ciertas desavenencias con su creador, Larry McVoy, llevaron a Linus Torvalds a decantarse por buscar un nuevo sistema de control de versiones.
b. ¿A quién pertenece actualmente Github y por qué?
    El 4 de junio de 2018 Microsoft compró GitHub por la cantidad de 7500 millones de dólares, la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios.
