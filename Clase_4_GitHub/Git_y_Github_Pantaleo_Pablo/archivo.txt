a. ¿Quién inventó el sistema de control de versión Git y por qué?

    Git fue inventado por Linus Torvalds para el desarrollo y versionado del codigo del kernel de Linux.


b. ¿A quién pertenece actualmente GitHub y por qué?

    Github pertenece actualmente a Microsoft, porque lo adquirió el 4 de Junio del 2018.

c. ¿Hay otra forma que no sea la terminal para trabajar con GitHub?

    Sí, por ejemplo la extensión de VSCode ofrece una interfaz gráfica para trabajar con GitHub.
