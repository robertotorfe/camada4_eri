1. ¿Quién inventó el sistema de control de versión Git y por qué?

Git se origina desde la necesidad de los desarrolladores de Linux para trabajar de manera colaborativa. Éste sistema nace luego de la caida de BitKeeper, un servicio de DVCS. El sistema principal utilizado por los desarrolladores del Kernel de Linux.

Linus Tovaldi (creador de Linux) fue uno de los principales interesados en impulsar el proyecto desde el 2005, aprovechando los aprendizajes del fallo de BitKeeper. Para ello, los requerimientos principales de este sistema fueron

  • Velocidad
  • Diseño Sencillo
  • Gran soporte para desarrollo no lineal (miles de ramas paralelas)
  • Completamente distribuido
  • Capaz de manejar grandes proyectos (como el kernel de Linux) eficientemente.

2. ¿A quién pertenece actualmente Github y por qué?

El 4 de junio de 2018 Microsoft compró GitHub por la cantidad de 7500 millones de dólares​, al inicio el cambio de propietario generó preocupaciones y la salida de algunos proyectos de este repositorio​, sin embargo no fueron representativos. GitHub continúa siendo la plataforma más importante de colaboración para proyectos Open Source.
